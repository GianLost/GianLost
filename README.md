## 📌 Sobre mim

Sou um profissional da área de tecnologia com atuação voltada ao **desenvolvimento de software**, com foco principal no ecossistema **.NET**. Tenho experiência no desenvolvimento de **APIs**, **aplicações web com ASP.NET Core**, interfaces modernas com **Blazor/MudBlazor** e soluções voltadas à **organização, automação e melhoria de processos**.
